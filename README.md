# Learning OpenGL from TheCherno
Learning OpenGL from the TheCherno's tutorial playlist
- Note: I'm using GLAD instead of GLEW

[TheCherno's Playlist](https://youtube.com/playlist?list=PLlrATfBNZ98foTJPJ_Ev03o2oq3-GGOS2&si=vj1Pc8lB6RYWT9om)

## Build
(Run from root repo directory)

```bash
   cmake -S . -B build
   cmake --build build
   ./build/bin/LearningOpenGL
```
(or `build\bin\LearningOpenGL.exe` for windows)
